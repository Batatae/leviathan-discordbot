# Leviathan Discord Bot

Discord bot developed in Python with multiple interactive commands.

## Features
- Interactive games (hangman, anagram, minesweeper)
- Custom commands
- Modular structure using cogs

## Tech
- Python
- discord.py

## Setup

1. Clone the repository:
git clone https://github.com/Batatae/leviathan-discordbot.git

2. Install dependencies:
pip install -r requirements.txt

3. Create a `.env` file in the root folder and add your token:
TOKEN=your_token_here

4. Run the bot:
python main.py
